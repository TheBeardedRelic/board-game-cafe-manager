# Board Game Café Manager

A Java-based application for managing a board-game café: games, customers, tables, and bookings.  
Built as a long-term learning and portfolio project to develop real-world software engineering skills.

---

## MVP Goals
- Manage games (add, list, search)
- Manage customers
- Manage tables
- Create bookings with double-booking prevention
- JSON persistence
- Console-based UI

---

## Planned Architecture
- **Model layer:** Game, Customer, Table, Booking  
- **Manager/controller layer:** central logic  
- **Persistence layer:** JSON reader/writer (later upgraded to SQL)  
- **UI layer:** console menus (later upgraded to JavaFX GUI)

---

## Future Phases
1. **SQL Upgrade** – SQLite database + DAO classes  
2. **GUI Upgrade** – JavaFX interface  
3. **Advanced Features** – ratings, recommendations, analytics  
4. **Optional REST API** – for web or mobile integration

---

## Purpose
This project is designed to:
- Build real-world development habits  
- Practice architecture and system design  
- Create a portfolio-ready application  
- Support long-term expansion and learning
